##配置本机ssh-key

  • 1.本机生成ssh-key
  • 2.查看本地私钥文件
  • 3.登陆Gitlab找到Settings配置选项

  • 4.配置ssh-key访问
      1. 选择SSH-Keys选项
      1. 在Add an SSH key中添加刚刚在本机生成的ssh-key
      1. 填写ssh-key的名称
      1. 添加ssh-key

  • 5.回到项目主页选择ssh链接

3.数据库

3.9 数据库基本概念

1、关系：两个表的公共字段

2、行：也称记录，也称实体

3、列：也称字段，也称属性

4、数据冗余：相同的数据存储在不同的地方

5、数据完整性：正确性+准确性=数据完整性

思考：学生的年龄是整型，输入1000岁，正确性和准确性如何？

答：正确的，但不准确

思考：年龄是整形的，收入了字符串，正确性和准确性如何？

答：不正确

3.11 数据规范化

Codd博士定义了6个范式来规范化数据库，范式由小到大来约束，范式越高冗余越小，但表的个数也越多。实验证明，三范式是性价比最高的。
